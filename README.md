# Laravel App (Breeze + Alpine)

A clean Laravel starter using **Laravel Breeze**, **Blade**, and **Alpine.js**.  
No Jetstream. No Livewire.

## Stack

- Laravel
- Laravel Breeze (Blade)
- Alpine.js
- Vite
- PHPUnit / Pest

## Requirements

- PHP ^8.1
- Composer
- Node.js + npm
- MySQL (local dev)

## Setup

## bash

composer install
cp .env.example .env
php artisan key:generate
npm install
npm run dev
php artisan migrate

## Frontend

Blade
Alpine.js
Vite
